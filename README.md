# To do list

It's a website/ webpage where you can do your task management i.e it's a task management website

It's made using *React* and *vite* .

It features a smooth and give animated UI


 Live demo - 


## Features 
- Add task 
- delete task
- responsive design 
- fast performance 


## Used tech
- React 
- vite
- framer motion
- lucid react
